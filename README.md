# GymZone

GymZone je tmavy web pre fitko s treningami, galeriou, cennikom, kontaktnym formularom a nacitanim dat z JSON suboru.

## Ako spustit web

1. Nainstaluj zavislosti:

```bash
npm install
```

2. Spusti lokalny server:

```bash
npm run dev
```

3. V prehliadaci otvor:

```text
http://localhost:5173/zadanie/index.html
```
